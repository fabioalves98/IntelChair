What is the IntelChair?

Our project consists of a motorized wheelchair that can autonomously move itself in a known enviroment, giving the user the ability of handsfree mobility while sitting in the chair. For more details visit the client page.

View client

Who will use it?

With this project we don't have a particular target audience. Anyone can use our chair, but it is possible to find more information on use cases and scenarios in the documentation page.

View vision

How will it be used?

The IntelChair will be used in conjunction with a mobile application with which the user can control all of its functions. For more information visit the documentation page.
